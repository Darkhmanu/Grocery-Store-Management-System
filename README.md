# Grocery Store SQL Analysis

A compact SQL project analyzing a Grocery Store Management System using MySQL.
Includes customer behavior, product performance, sales trends, supplier analysis, and employee efficiency.

## Files
- Categories.csv
- Customers.csv
- Employees.csv
- Products.csv
- Suppliers.csv
- Orders.csv
- OrderDetails.csv
- Grocery_Store_Management.pdf
- README.txt

## Database Tables
- supplier
- categories
- employees
- customers
- products
- orders
- order_details

## Steps
1. Created MySQL database & schema
2. Imported all CSVs using Table Data Import Wizard
3. Cleaned order_date and converted to DATE format
4. Ran analysis queries for:
   - Customer Insights
   - Product Performance
   - Sales Trends
   - Supplier Contribution
   - Employee Performance
   - Order Details

## Key Insights
- 156 unique customers
- High-value customers drive most revenue
- Strong seasonal sales trends
- Weekend orders outperform weekdays
- Supplier revenue varies significantly

## Conclusion
A complete SQL analysis project demonstrating schema creation, data import, cleaning, joins, aggregations, and business insights using MySQL.
